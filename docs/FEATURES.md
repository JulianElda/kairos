# Features

- Show current location name, current temperature, and weather description.
- Show weather data for the next couple of hours.
- Show weather data for the next couple of days.
- Use browser geolocation to refresh weather data after initial server-side render.
- Display sunrise/sunset, UV index, humidity, wind speed, and apparent temperature.
- Dark mode with a footer toggle that remembers your preference and avoids flashing on load.
- Contextual weather icons that visually represent current conditions; each icon has day and night variants and is selected from the reported weather code.
